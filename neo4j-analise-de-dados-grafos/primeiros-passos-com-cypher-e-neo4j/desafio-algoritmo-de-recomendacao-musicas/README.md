# 📊 Desafio: Criando um algoritmo de recomendação de músicas com base em grafos

O desafio consiste em desenvolver um sistema de recomendação de músicas que utiliza grafos para identificar padrões de escuta e sugerir novas faixas aos usuários.

## 🎯 Objetivos

Modelar e criar um grafo de recomendação de músicas. O sistema deve ser capaz de:

- **Representar (nós)**: Usuarios, Músicas, Artistas, Generos.
- **Conexões (relacionamentos)**: ESCUTAR, CURTIR, SEGUIR
- **Consultas Cypher**: Utilizar consultas para gerar recomendações personalizadas com base em conexões no grafo.
