# TOTVS - fundamentos de engenharia de dados e machine learning

> Um bootcamp para construir uma base sólida em Python e banco de dados, entender os fundamentos de ETL e cloud, evoluir até o Machine Learning - aplicando cada conceito em projetos que simulam desafios reais do mercado de dados.
-------------

## Atividades

- Desafio: Criando um dashboard de vendas do xbox com Excel;
- Desafio: Decisão automática para pedidos;
- Desafio: Operadores estruturais condicionais;
- Desafio: Redução de custos farmácia com a AWS;
- Instalação MongDB via docker e inserção de colletions no bd via python;
- Fundamentos de ETL com Python: Foram usadas bibliotecas Pandas e suas funções, a Scikit-Learn e Framework Luigi para ETL;
- Desafio: Explorando IA Generativa em um pipeline de ETL com Python;
