## Bootcamp Bradesco GenAI e Dados

Esse repositório contém os códigos utilizados durante o bootcamp do bradesco, onde foram abordados temas relacionados a ciência de dados, inteligência artificial e machine learning. O objetivo é compartilhar conhecimento.

## Conteúdo

- **Data base**:

  Conhecer os principais comandos de manipulação de tabelas.

- **GenAI**:

    Criar script para GenAI para criar db com informações relacionais sobre pcg pokemon.
